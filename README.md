# ArkWallets - ARK Desktop Wallet plugin
Use ArkWallets.com directly within the wallet

## Installation

 1. Clone this repository into `~/.ark-desktop/plugins/`.
 2. Start the desktop wallet.
 3. Enable the plugin on the plugin section.
